# Expense Tracker

A full-stack web application for tracking personal expenses.

## Features
- User authentication
- Expense CRUD operations
- Category-based summaries
- Responsive design

## Tech Stack
- Frontend: React + TypeScript
- Backend: Node.js + Express + TypeScript
- Database: PostgreSQL
