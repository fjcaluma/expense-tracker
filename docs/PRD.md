# Product Requirements Document: Expense Tracker

## Problem Statement
People struggle to understand where their money goes each month, leading to poor financial decisions.

## Target Users
- Young professionals tracking personal expenses
- Anyone wanting to understand spending patterns

## Core Features (MVP)
1. User authentication (signup/login)
2. Add/edit/delete expenses
3. Categorize expenses (food, transport, entertainment, etc.)
4. View list of all expenses
5. See total spending by category (this month)

## Nice-to-Haves (Post MVP)
- Date range filtering
- Charts/graphs
- Budget limits with alerts
- Receipt photo uploads
- Export to CSV

## Success Metrics
- User can add expense in <10 seconds
- All data persists between sessions
- App loads in <2 seconds

## Technical Constraints
- Must work on mobile browsers
- Free hosting (under $0/month initially)
